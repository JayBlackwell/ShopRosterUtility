# Golf Club Roster Merger

A Streamlit application that helps merge duplicate member profiles by copying Member Card IDs from profiles that have them to matching profiles that don't, then removing the source profiles.

## Features
- Automatically detects and handles duplicate member profiles
- Preserves ID formatting to prevent scientific notation
- Provides detailed statistics and visualizations of changes
- Easy-to-use web interface
